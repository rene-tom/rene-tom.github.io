Mini-shell
11/2022 - 01/2023
C

In my 3rd year of bachelor, we did a copy of a shell using C by groups of 3.
We had to make the commands pwd,cd and exit ourselves.
We also had to implement redirects ">" , pipes « | » and the ineterpretation of the symbol « * ».

I've worked on :
- The cd/pwd implemntation
- The variables implementations
- Pipes
- I've also helped with some cases with the "*" implementation
It was an interesting project because it was dense and we had to do it in a short amount of time
The work distributuin was a key component in the success of this project
