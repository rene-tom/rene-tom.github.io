Dactylogame
11/2022 - 01/2023
Java

En troisième année de licence, nous avons dû par binôme réaliser un autre
jeu sur Java. C'est un jeu d'écriture sur clavier inspiré de sites comme
10fastfingers ou monkey-type.

Dans ce projet je me suis occupé de :
- La lecture des inputs
- Les fonctionnalités du mode « Entraînement »
( Choix du mode temps ou nombre de mots, statistiques de fin de jeu, UI )
- Les fonctionnalités du mode « Solo » ( Niveau de départ, Timer gestion de la vie)
- J'ai aidé sur la partie réseau (Communication des touches effectuées)

Ce projet était intéressant car c'était le premier projet en réseau que j'ai fait
