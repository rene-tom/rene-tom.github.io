Witch Madness
01/2024 - 02/2024
Unreal Engine / C++

Le jeu est jouable ICI

Ceci est mon troisième projet fait tout seul
Il a été réalisé en l'espace d'un mois

C'est un jeu d'arcade de type automatic runner (comme Subway Surfers ou Temple run par exemple)
Le but est de battre les sorcières qui se dressent sur notre chemin et de récupérer les améliorations dans le but de battre le boss de fin.
Ce jeu possède 2 personnages jouable, qui possèdent des sorts différents.
J'ai tout fait moi-même dans le jeu, que cela soit la programmation ou bien les assets et fx

J'ai été en charge de l'implémentation tous les éléments de gameplay, dont :

  • - L'apparition semi-aléatoire des ennemis
  • - Les mécaniques de déplacement (avancement automatique, déplacement latéral, téléportation )
  • - Les différentes améliorations qu'obtient le joueur dans la partie
  • - L'intégration du Son
